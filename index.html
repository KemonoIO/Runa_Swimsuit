Runa Swimsuit

Summary

This package is a shiny swimsuit for the Runa model by Torinyan. You must own the Runa base model to use it.

Installation

  1. Unpack the dependencies, in order.
  2. Unpack Runa_Swimsuit.unitypackage.
  3. Drag Assets/Kemono_Io/Runa_Swimsuit.prefab into the Runa_Source object in the Hierarchy panel.
  4. Set the Runa_Swimsuit's anchor override.
    1. Select the Runa_Swimsuit mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
  5. Constrain the Runa_Swimsuit's armature to the Runa base model's armature.
    1. Fully expand the Runa_Swimsuit object's armature in the Hierarchy panel.
    2. Select one of the bones, such as hips.
    3. Drag the identically-named bone from the Runa base model's armature to the Parent Constraint field in the Inspector panel.
    4. Click the Activate button.
    5. Repeat the last three steps for each bone in the armature.
  6. Add the swimsuit to the breathe animation.
    1. Select the Runa_Source object in the Hierarchy panel.
    2. Set the Animator's Controller to FX_Runa in the Inspector panel.
    3. Select the Animation tab and click the lock icon if it is not already locked.
    4. Select the Runa_Swimsuit object in the Hierarchy panel.
    5. Select the breathe animation and click the record button.
    6. Select the first keyframe.
    7. Set the Breathe blend shape to 0.
    8. Select the second keyframe.
    9. Set the Breathe blend shape to 100.
    10. Select the third keyframe.
    11. Set the Breathe blend shape to 0.
    12. The timeline should look as follows.
    13. Click the Preview button to exit record mode.
    14. Click the lock icon.
    15. Right click the Animator's Controller and select Revert.
  7. Optionally, the transparency and color of the swimsuit can be altered.
    1. Select the Runa_Swimsuit object in the Hierarchy panel.
    2. To change transparency, alter the Alpha Mod property in the Inspector Panel to a value between 0 and -1, where 0 is full opacity and -1 is full transparency.
    3. To shift color, alter the Hue Shift property in the Inspector panel.
    4. To change color to black, set the Saturation property to -1 and the Brightness property to -0.2.
    5. To change color to white, set the Saturation property to -1 and the Brightness property to 0.4.
